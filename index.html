擴展網路

Sophos Central 支持輕鬆部署新防火牆、SD-RED、無線和所有 Sophos 產品。

  • 從 Sophos Central 將新 Sophos Firewall 部署到遠端地點，無需接觸。

  • 部署零接觸 Sophos SD-RED 分支辦事處、家庭辦公室或 ICS 自動配置 VPN 設備，只需點擊幾次滑鼠。

  • 輕鬆添加新的隨插即用 Sophos APX 無線接入點。

擴大您的保護

Sophos Central 能夠擴充您的團隊實力，支持您通過單一控制台管理防火牆、端點、伺服器、行動裝置和其他 Sophos 產品，同時發揮 Synchronized Security 整合的優勢。

  • Security Heartbeat™ 安全心跳與您的防火牆即時共享端點運行狀況。

  • 自動回應隔離受威脅系統，阻止威脅橫向轉移到裝置和資料丟失。

  • 同步應用程式控制可識別 100% 經加密和規避的應用程式。

  • 同步用戶 ID 方便與 Active Directory 整合。
